Get started fast

MapBox.js is hosted on the same reliable and speedy CDN that powers MapBox tiles. There's no API key to use it. All that's needed is a reference to the script & CSS in your HTML.

<link href='//api.tiles.mapbox.com/mapbox.js/{{site.mapboxjs}}/mapbox.css' rel='stylesheet' />

MapBox.js is built on top of Leaflet — an open-source mapping library.

The source code is open and available on GitHub and takes advantage of open specifications like TileJSON, UTFGrid and GeoJSON.

MapBox.js is browser tested & supported. Compatible with IE8+ and all modern browsers.

MapBox.js is optimized for mobile devices and small screens by default. For additional features, check out the Mobile section.

Have a question? Try StackOverflow. We're also happy to help out with code on support.mapbox.com.
